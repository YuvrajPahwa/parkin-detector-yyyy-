# 🚗 AI-Powered Smart Parking Detection

An intelligent parking detection system that uses AI and computer vision to determine whether a parking space is **Free** or **Occupied** from uploaded images.

---

## 📌 Overview

Parking management is a major challenge in smart cities. This project provides a simple and efficient solution using **AI-based image processing** to detect parking availability in real time.

Users can upload an image through a web interface, and the system analyzes it to determine parking status.

---

## ✨ Features

- 📷 Upload parking images
- 🧠 AI-based detection
- ⚡ Fast and lightweight processing
- 🌐 Simple web interface using Flask
- 📊 Output: **Free / Occupied**

---

## 🛠️ Tech Stack

- **Python**
- **Flask** – Web framework
- **OpenCV** – Image processing
- **NumPy** – Numerical operations
- **YOLOv8** – Object detection (Ultralytics)
